```javascript
async function apiFetch(url, method, body) {
  const opts = {
    method,
    headers: { 'X-Session-ID': sessionId, 'Content-Type': 'application/json' }
  };
  if (body) opts.body = JSON.stringify(body);
  const res = await fetch(url, opts);
  const data = await res.json();
  if (!res.ok) { const err = new Error(data.error || 'API error'); err.status = res.status; throw err; }
  return data;
}
```

**my-hotels-module.js:**
Create at `agent-network/public/modules/my-hotels-module.js`. Structure:

1. **State:** `let sessionId = ''`, `let hotels = []`

2. **init():** Called on DOMContentLoaded.
   - Read `sessionId` from `localStorage.getItem('agent_session_id')`
   - If no sessionId: call `showAuthGate('Prisijunkite, kad galetumete valdyti savo viesbucius.')` and return
   - Otherwise: call `loadMyHotels()`

3. **apiFetch(url, method, body):** Copy exact pattern from hotel-admin-module.js (sends X-Session-ID header, parses JSON, throws on !res.ok)

4. **loadMyHotels():**
   - `const data = await apiFetch('/api/hotels/mine', 'GET')`
   - `hotels = data.hotels`
   - If hotels.length === 0: render empty state with create form prominent
   - If hotels.length > 0: render hotel list + "Prideti viesbuti" button that shows create form
   - If hotels.length >= 3: hide create form/button, show "Pasiektas limitas (3 viesbuciai)" message

5. **renderHotelList():** For each hotel, render a card with:
   - Hotel name (h3)
   - City (small text, muted)
   - Created date (formatted as YYYY-MM-DD)
   - "Valdyti" link: `href="/hotel/${hotel.slug}/admin"` (primary button style)
   - "Perziureti" link: `href="/hotel/${hotel.slug}"` `target="_blank"` (secondary/outline button)

6. **showCreateForm():** Inline form (not modal) with fields:
   - Pavadinimas (name) — required, text input
   - Miestas (city) — text input, default "Palanga"
   - Adresas (address) — text input
   - Telefonas (phone) — text input
   - WhatsApp — text input
   - Submit button: "Sukurti viesbuti"
   - Cancel button: hides form (only shown when hotels.length > 0)

7. **handleCreate(e):**
   - e.preventDefault()
   - Read form values, validate name is non-empty
   - `const data = await apiFetch('/api/hotels', 'POST', { name, city, address, phone, whatsapp })`
   - On success: `window.location.href = '/hotel/${data.slug}/admin'`
   - On error: show error message below form (especially handle the "Maksimalus" limit error)

8. **showAuthGate(msg):** Render into `#my-hotels-root`:
   - CSS class `auth-gate`
   - Message text
   - Link to `/#login` or `/` with text "Prisijungti"

9. **Theme toggle:** Copy the same toggle pattern from hotel-admin-module (toggles `data-theme` attribute and saves to `localStorage.hotel_theme`)

10. **DOMContentLoaded:** `document.addEventListener('DOMContentLoaded', init)`

Do NOT use any framework, jQuery, or external CSS. Pure vanilla JS + inline styles in the HTML file. This matches the project standard.
