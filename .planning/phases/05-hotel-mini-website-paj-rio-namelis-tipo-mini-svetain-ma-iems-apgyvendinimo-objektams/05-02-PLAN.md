     - Triuksmas: `{ quiet: 'Ramus', moderate: 'Vidutinis', lively: 'Gyvas' }`
     - Only render rows where data exists (skip null/undefined)
   - Room description if present
   - Thumbnail gallery strip: horizontal scroll, 60x40px thumbnails, click swaps main photo
   - "Rezervuoti" button (background: #FFC700, color: #000, min-height: 44px)

   **getAvailabilityDisplay(room):**
   Copy from RESEARCH.md. Returns:
   - available -> `{ color: '#22c55e', label: 'LAISVAS', cssClass: 'available' }`
   - occupied + available_from within 14 days -> `{ color: '#eab308', label: 'Laisvas netrukus -- nuo {date}', cssClass: 'soon' }`
   - occupied + available_from > 14 days -> `{ color: '#ef4444', label: 'UZIMTAS / Laisvas nuo: {date}', cssClass: 'occupied' }`
   - occupied no date -> `{ color: '#ef4444', label: 'UZIMTAS', cssClass: 'occupied' }`
   - Dates formatted with `toLocaleDateString('lt-LT')`

   **Reservation form (inline, no modal):**
   - Click "Rezervuoti" -> slide open form below room card content (toggle class)
   - Form fields: vardas (text, required), atvykimas (date, required), isvykimas (date, required), zinute (textarea, 3 rows, optional)
   - Validation: all required fields -> show inline error in #ef4444 if missing
   - Two CTA buttons side by side:
     - "Siusti per WhatsApp" (background #25D366): `buildWhatsAppLink(hotel.whatsapp, buildReservationMessage(hotel, room, formData))` -> `window.open(url, '_blank')`
     - "Siusti el. pastu" (background #333): `buildMailtoLink(hotel.phone, hotel, room, formData)` -> `window.location.href = url`
       **NOTE: The hotels schema has a `phone` field (TEXT) which serves as the contact address for both phone and email. There is no separate `email` column. Use `hotel.phone` as the mailto recipient. If the value looks like a phone number rather than an email, the mailto link will still work but the user may need to update it in the admin panel.**
   - If hotel has no whatsapp -> hide WhatsApp button
   - If hotel has no phone -> hide email button and show contact info text instead

   **buildWhatsAppLink(whatsappNumber, messageText):**
   ```javascript
   const encoded = encodeURIComponent(messageText);
   return `https://wa.me/${whatsappNumber}?text=${encoded}`;
   ```

   **buildReservationMessage(hotel, room, formData):**
   ```javascript
   return `Sveiki! Noreciau rezervuoti ${room.name || 'kambari ' + room.number} viesbutyje "${hotel.name}".\n` +
     `Atvykimas: ${formData.arrival}\nIsvykimas: ${formData.departure}\n` +
     `Vardas: ${formData.name}\n${formData.message ? 'Pastaba: ' + formData.message : ''}`;
   ```

   **buildMailtoLink(email, hotel, room, formData):**
   ```javascript
   // email parameter comes from hotel.phone field (used as contact address)
   const subject = encodeURIComponent(`Rezervacija -- ${room.name || 'Kambarys ' + room.number}, ${hotel.name}`);
   const body = encodeURIComponent(buildReservationMessage(hotel, room, formData));
   return `mailto:${email}?subject=${subject}&body=${body}`;
   ```
