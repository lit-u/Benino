| Property | Value |

## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gap, photo badge inner padding, inline tag spacing |
| sm | 8px | Input field inner padding (vertical), between meta items, gallery item gap |
| md | 16px | Card inner padding, form field vertical rhythm, section header margin-bottom |
| lg | 24px | Between room cards on public page, between form fields groups |
| xl | 32px | Hotel section padding (top/bottom), between hotel header and room grid |
| 2xl | 48px | Page top margin below platform header (30px header + 18px breathing room) |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Platform header is fixed at 30px height — page content starts at `padding-top: 30px` (matches existing `body` rule in `public/style.css:17`)
- Touch-target minimum: 44px height for all interactive buttons (WhatsApp CTA, reservation submit, availability toggle)
- Photo drag handle grab area: 44×44px minimum

## Typography

Source: `public/style.css:8`, sizes verified against `modules-styles.css` and `listings-detail.css`.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label / Meta | 12px | 400 | 1.4 |
| Card title / Room name | 16px | 600 | 1.3 |
| Section heading / Hotel name | 20px | 700 | 1.2 |

Notes:
- 4 sizes declared — no additions permitted without checker re-approval.
- Weight 700 is used only for the hotel name `<h1>` on the public page and admin panel page title.
- Weight 600 is used for room names, CTA button labels, availability badge text.
- Weight 400 covers all body text, meta labels, form field values, description paragraphs.
- Sea-proximity info block uses 12px labels (role: Label/Meta) + 14px values (role: Body).

## Color

Source: `public/style.css:1-20` (dark background), `modules-styles.css:13` (brand accent `#FFC700`), `05-RESEARCH.md` (traffic light hex values confirmed against codebase pattern).

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#1a1a1a` | Page background, all surfaces that are not cards |
| Secondary (30%) | `#252525` | Room cards, admin panel sections, reservation form container, QR preview box |
| Accent (10%) | `#FFC700` | See reserved-for list below |
| Available (semantic) | `#22c55e` | Availability badge background when status = available only |
| Soon (semantic) | `#eab308` | Availability badge background when available_from within 14 days only |
| Occupied (semantic) | `#ef4444` | Availability badge background when status = occupied only |
| Destructive | `#ef4444` | Delete room button, delete hotel button — these are the only two destructive actions in this phase |

Accent `#FFC700` reserved for:
- "Rezervuoti" (Reserve) primary CTA button — background fill
- "Window view badge" (`🪟 Lango vaizdas`) pill on photo overlay
- Admin panel section headers and dividers (matches existing modal header pattern at `modules-styles.css:226`)
- Hover border on room cards (matches existing `.favorite-card:hover` pattern at `modules-styles.css:134-136`)
- Focus ring on form inputs

Accent is NOT used for: body text, availability badges, secondary action buttons, error states, or any decorative background.

Destructive `#ef4444` is NOT used for the availability "occupied" state in the admin toggle — the toggle is a status indicator, not a destructive action. It only appears on delete confirm buttons.

## Component Specifications

### Public Page `/hotel/:slug`

**Hotel Header**
- Full-width section, `background: #252525`, `border-radius: 12px`, padding: `xl` (32px) top/bottom, `lg` (24px) left/right
- Hotel name: `<h1>`, 20px, weight 700, color `#f5f5f5`
- Address: `<address>`, 14px, weight 400, color `#9ca3af`
- Description: `<p>`, 14px, weight 400, color `#d1d5db`, max 3 lines with text-overflow ellipsis initially, "Daugiau" expand link in `#FFC700`

**Room Card**
- `<article>` element, `background: #252525`, `border-radius: 12px`, border `1px solid #333`
- Hover: border-color `#FFC700`, box-shadow `0 2px 8px rgba(255, 199, 0, 0.2)`, transition `0.2s`
- Photo area: top of card, `aspect-ratio: 3/2`, `overflow: hidden`, first photo always displayed
- Window view badge: absolute positioned top-left, `background: rgba(255, 199, 0, 0.9)`, color `#000`, 12px, weight 600, padding `4px 8px`, `border-radius: 0 0 8px 0`
- Availability badge: absolute positioned top-right, pill shape, `border-radius: 20px`, 12px, weight 600, padding `4px 12px`, color `#000` (on all three semantic background colors)
- Card body padding: `md` (16px) all sides
- Room name: 16px, weight 600, color `#f5f5f5`
- Price: 16px, weight 600, color `#FFC700`, format: `{N} €/naktis`

**Sea-Proximity Info Block**
- `background: #1a1a1a`, `border-radius: 8px`, `border: 1px solid #333`, padding `sm` (8px) `md` (16px)
- Three rows, each: emoji icon + 12px label + 14px value
  - Row 1: `🌊` + `Iki jūros` + `{N} min. pėsčiomis ({M}m)`
  - Row 2: `🧭` + `Orientacija` + `{compass} — {orientation_note if set}`
  - Row 3: `🔇` + `Triukšmas` + `Ramus` / `Vidutinis` / `Gyvas`
- Divider between rows: `border-bottom: 1px solid #2d2d2d`
- Absent values: row not rendered (do not show empty rows)

**Reservation Form**
- Triggered by "Rezervuoti" button click — slides in or replaces card content (no separate modal for MVP)
- Form container: `background: #252525`, `border-radius: 12px`, `border: 1px solid #444`, padding `md` (16px)
- Fields: vardas (text), atvykimas (date), išvykimas (date), žinutė (textarea, 3 rows)
- Input style: `background: #1a1a1a`, `border: 1px solid #444`, `border-radius: 8px`, color `#f5f5f5`, 14px, padding `8px 12px`
- Input focus: `border-color: #FFC700`, `outline: none`
- CTA buttons row (side by side):
  - WhatsApp button: `background: #25D366`, color `#fff`, `border-radius: 8px`, padding `12px 20px`, 14px weight 600, min-height 44px
  - Email button: `background: #333`, color `#f5f5f5`, `border: 1px solid #555`, `border-radius: 8px`, padding `12px 20px`, 14px weight 600, min-height 44px
- Buttons gap: `sm` (8px)

**Room Gallery (within card, secondary photos)**
- Horizontal scroll strip below main photo: `display: flex`, `gap: 8px`, `overflow-x: auto`, `padding: 8px`
- Thumbnail size: `60×40px`, `border-radius: 6px`, `object-fit: cover`, `cursor: pointer`
- Active thumbnail: `border: 2px solid #FFC700`
- Scrollbar hidden: `scrollbar-width: none`

### Admin Page `/hotel/:slug/admin`

**Page Layout**
- Same dark background `#1a1a1a`
- Top section: hotel name (20px, 700) + "Valdymo Panelė" sub-label (12px, `#9ca3af`)
- Two-column layout at 768px+ breakpoint: room list (left, 300px fixed) + room editor (right, flex-1)
- Below 768px: stacked single column

**Room List (Admin Sidebar)**
- Each room row: `background: #252525`, `border-radius: 8px`, `border: 1px solid #333`, padding `sm 8px md 16px`
- Active room: `border-color: #FFC700`
- Room row content: room number badge (12px, `background: #333`, `border-radius: 4px`, padding `2px 6px`) + room name (14px) + availability dot (8×8px circle in semantic color)
- "Pridėti kambarį" button at list bottom: `background: transparent`, `border: 1px dashed #555`, color `#9ca3af`, 14px, full-width, `border-radius: 8px`, padding `md 16px`, min-height 44px
- Disabled when rooms === 10 (MVP limit): color `#555`, cursor `not-allowed`

**Room Editor (Admin Main Panel)**
- Sections separated by `border-top: 1px solid #2d2d2d` + section label (12px, `#9ca3af`, uppercase, letter-spacing 0.1em)
- Form inputs: same style as public reservation form inputs
- Orientation dropdown: standard `<select>`, same input style, options: N / NE / E / SE / S / SW / W / NW (stored values), display names: Šiaurė / Šiaurės Rytai / Rytai / Pietryčiai / Pietūs / Pietvakariai / Vakarai / Šiaurės Vakarai
- Noise dropdown: options Ramus / Vidutinis / Gyvas
- Availability toggle: 3-state radio-button group styled as pill toggle
  - Pill 1: `background: #22c55e`, text "LAISVAS" — color `#000`
  - Pill 2: `background: #eab308`, text "NETRUKUS" — color `#000`
  - Pill 3: `background: #ef4444`, text "UŽIMTAS" — color `#fff`
  - Active pill: full background color; inactive: `background: #333`, color `#9ca3af`
  - "Laisvas nuo" date input appears below toggle only when NETRUKUS or UŽIMTAS is active
- Save button: `background: #FFC700`, color `#000`, weight 600, `border-radius: 8px`, full-width, min-height 44px, text "Išsaugoti"

**Photo Manager (Drag-Drop)**
- Photo grid: `display: grid`, `grid-template-columns: repeat(3, 1fr)`, `gap: sm (8px)`
- Photo item: `aspect-ratio: 3/2`, `border-radius: 8px`, `overflow: hidden`, `cursor: grab`, `border: 2px solid transparent`
- Window view photo item: `border-color: #FFC700`
- Dragging state: `opacity: 0.5`, `cursor: grabbing`
- Drop target: `border: 2px dashed #FFC700`
- Photo overlay on hover: semi-transparent `rgba(0,0,0,0.6)` with icon row: trash icon (`#ef4444`) + window-view toggle icon (`#FFC700`)
- "Lango vaizdas" toggle: checkbox-style, label "Lango vaizdas", 12px — only one photo can have this designation; toggling on another auto-removes from previous
- Upload zone: dashed border `#555`, `border-radius: 8px`, padding `xl 32px`, text "Vilkite nuotraukas arba spustelėkite" (14px, `#9ca3af`), `background: #252525`

**QR Code Section (per room)**
- QR preview: inline `<img>` from dataURL, 120×120px, `border-radius: 8px`, `background: #fff` (QR requires white background), padding `sm 8px`
- Room URL display: 12px, `#9ca3af`, monospace (`font-family: 'Courier New', monospace`), truncated
- Action buttons side by side:
  - "Atsisiųsti QR": `background: #333`, `border: 1px solid #555`, color `#f5f5f5`, 14px, `border-radius: 8px`, padding `8px 16px`, min-height 44px
  - "Kopijuoti URL": same style, color changes to `#22c55e` for 2s after copy (success feedback)

## Interaction States

| State | Specification |
|-------|---------------|
| Button :hover | Brightness +10% via `filter: brightness(1.1)`, transition `0.15s` |
| Button :active | `transform: scale(0.98)`, transition `0.1s` |
| Button :disabled | `opacity: 0.4`, `cursor: not-allowed`, no hover effect |
| Form input :focus | `border-color: #FFC700`, `box-shadow: 0 0 0 2px rgba(255, 199, 0, 0.2)` |
| Photo drag :dragging | `opacity: 0.5`, `transform: rotate(2deg)` |
| Copy URL success | Button text changes to "Nukopijuota!" for 2000ms, then reverts |
| Availability save success | Section flashes `background: rgba(34, 197, 94, 0.1)` for 1000ms |
| Error (API failure) | Inline error message below the triggering element, 12px, color `#ef4444`, no modal |
