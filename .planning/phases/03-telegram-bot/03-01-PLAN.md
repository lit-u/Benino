Add a "telegram" key after the "scoring" block (before "sources"):

```json

  "batchDelayMs": 1000,
  "dispatchCutoffMs": 86400000
}
```

Explanation:
- batchSize: 10 items max per dispatch run
- batchDelayMs: 1000ms between individual sends (Telegram rate: 30/sec to same chat; 1/sec is safe)
- dispatchCutoffMs: 86400000 = 24h — only send items scored within last 24h on first run
