# Hero's Journey — Herojaus Kelionė

## Kas tai

Joseph Campbell "Hero with a Thousand Faces" + Christopher Vogler "The Writer's Journey" adaptacija. Universali archetipinė struktūra — veikia visuose žanruose ir kultūrose.

## 12 Etapai (Vogler versija)

| # | Etapas | Aprašymas |
|---|---|---|
| 1 | **Ordinary World** | Herojaus kasdienis pasaulis prieš kelionę — kontrastas tam kas bus |
| 2 | **Call to Adventure** | Kvietimas — problemos, iššūkio ar galimybės forma |
| 3 | **Refusal of the Call** | Herojus baiminasi, atsisako — parodo žmogiškumą ir stakes |
| 4 | **Meeting the Mentor** | Senolis/mokytojas suteikia dovaną: žinių, įrankių, pasitikėjimo |
| 5 | **Crossing the Threshold** | Herojus palieka komforto zoną — point of no return |
| 6 | **Tests, Allies, Enemies** | Naujas pasaulis — kas draugas, kas priešas, kokie taisyklės |
| 7 | **Approach to the Inmost Cave** | Artėjimas prie pagrindinio pavojaus — pasiruošimas |
| 8 | **Ordeal** | Didžiausia krizė — herojus simboliškai miršta ir atgimsta |
| 9 | **Reward (Seizing the Sword)** | Pergalė — herojus gauna tai ko ieškojo (ar mano kad gauna) |
| 10 | **The Road Back** | Grįžimas — bet antagonizmas dar nesibaigia |
| 11 | **Resurrection** | Paskutinis išbandymas — herojus turi įrodyti transformaciją |
| 12 | **Return with the Elixir** | Grįžimas į Ordinary World su dovana visiems |

## Archetipai (personažų funkcijos)

| Arhetipas | Funkcija | Pavyzdžiai |
|---|---|---|
| **Hero** | Transformacija, mokymasis | Protagonistas |
| **Mentor** | Motyvacija, mokymas, dovanos | Yoda, Gandalf, mokytojas |
| **Threshold Guardian** | Testuoja herojaus ryžtą | Sargybinis, biurokratas |
| **Herald** | Skelbia pokyčio būtinybę | Žinianešys, įvykis |
| **Shapeshifter** | Kelia abejonių, nestabilus | Meilės partneris, sąjungininkas? |
| **Shadow** | Pagrindinė tamsioji jėga | Antagonistas, vidinis demonas |
| **Trickster** | Komiška pagalba, status quo ardymas | Komiškas draugas |
| **Ally** | Palaikymas, specifiniai gebėjimai | Draugas, komanda |

## Kombinuotas naudojimas

```
Hero's Journey etapai → Save the Cat beatai → McKee scenos
       (makro)                (mezo)              (mikro)
```

Pavyzdys:
- HJ Etapas 8 (Ordeal) = STC Beatas 11 (All Is Lost)
- HJ Etapas 1 (Ordinary World) = STC Beatai 1–3 (Opening Image + Set-Up)

## Kaip naudoti su audio medžiaga

1. **Identifikuok Ordinary World** — koks gyvenimas buvo "prieš"
2. **Rask Call to Adventure** — kas pakeitė viską
3. **Identifikuok Mentor figūrą** — kas padėjo, mokė, davė įrankių
4. **Ordeal** — koks buvo tikrasis lūžio taškas
5. **Elixir** — ką herojus grąžina pasauliui po kelionės
